CS4465 Computational Biology and Bioinformatics — Ultimate Study Guide

🧬 1. Introduction to Molecular Biology | 分子生物學入門

Body Structure Hierarchy | 身體結構層次

Body → Organs → Tissues → Cells
(身體 → 器官 → 組織 → 細胞)

Key Molecules | 關鍵分子

  1. DNA: Stores genetic info. (儲存遺傳信息)
  2. RNA: Intermediate between DNA and proteins. (DNA 同蛋白質之間嘅中介)
  3. Protein: Performs chemical reactions. (執行化學反應)

Central Dogma | 分子生物學中心教條

DNA→Transcription\xrightarrow{\text{Transcription}}Transcription​ mRNA→Translation\xrightarrow{\text{Translation}}Translation​ Protein

  • Transcription (轉錄): DNA 變 mRNA。
  • Translation (翻譯): mRNA 變蛋白質。

[!TIP]
💡 Memory Master (廣東話口訣):
「一條龍服務」: DNA 係藍圖 (Blueprint)，RNA 係影印本 (Photocopy)，Protein 係製成品 (Product)。

🔍 2. Sequence Similarity and Alignment | 序列比對

Needleman-Wunsch (Global Alignment) | 全域比對

Align entire sequences. (成條序列由頭到尾對齊)

  • Recurrence:V(i,j)=max⁡{Diagonal + match, Up + gap, Left + gap}V(i,j) = \max \{ \text{Diagonal + match, Up + gap, Left + gap} \}V(i,j)=max{Diagonal + match, Up + gap, Left + gap}

Smith-Waterman (Local Alignment) | 區域比對

Align subsequences with highest similarity. (淨係搵最似嗰段嚟比對)

  • Key: Adds a 0 to the max function (No negative scores).

[!TIP]
💡 Memory Master (廣東話口訣):

  • Needleman-Wunsch: 「全份做晒」。似考 Essay，由頭寫到尾，一格都唔放過。
  • Smith-Waterman: 「重點觀察」。似睇重點，唔好嘅地方直接俾 0 分唔計，淨係睇最夾嗰段。

🔎 3. Database Search (BLAST) | 數據庫搜索

BLAST: Heuristic search (Fast but not 100% optimal).
(啟發式搜索：快，但未必係最準，係「攞個大概」。)

BLAST Variants | BLAST 版本

  • Blastp: Protein→\to→ Protein
  • Blastn: DNA→\to→ DNA
  • Blastx: Translated DNA→\to→ Protein

[!TIP]
💡 Memory Master (廣東話口訣):
「快閃搜尋」: BLAST = Be Like A Speedy Tool。用 E-value 嚟睇結果，「E 值越細，緣份越深」（愈細越 Significant）。

🧩 4. Multiple Sequence Alignment (MSA) | 多序列比對

Methods | 方法

  1. Center Star: Choose a “star” sequence and align others to it. (搵一個中心點，其他圍住佢嚟對)
  2. ClustalW: Progressive alignment using a tree. (由親戚開始，一對對咁加上去)
  3. MUSCLE: Iterative refinement. (做完一次再執，執到最靚為止)

[!TIP]
💡 Memory Master (廣東話口訣):
「排隊排齊」: Center Star 係「排隊跟大佬」，ClustalW 係「逐個加入」，MUSCLE 係「執完又執」。

🌳 5. Phylogeny Reconstruction | 系統發育重建

Methods | 方法

  • Maximum Parsimony: Minimum mutations (The simplest explanation is best).
    • (最簡約法：搵最少變異嘅樹，最簡單就係最好。)
  • UPGMA: Distance-based, assumes constant rate (Molecular clock).
    • (距離法：次次攞平均值去合體。)

[!TIP]
💡 Memory Master (廣東話口訣):
「搵祖宗」: Parsimony 係「慳得就慳」（少變異），UPGMA 係「夾份攞平均」（平均分）。

🔍 6. Motif Discovery | 基序發現

Motifs: Short conserved patterns (Transcription factor binding sites).
(基序：DNA 入面一啲短而保守嘅模式，通常係蛋白質開關嘅位置。)

Gibbs Sampling | 吉布斯採樣

Iterative probabilistic sampling to find motifs.
(隨機抽樣：每次抽一條出嚟重新估計，直到搵到最似樣嘅 Motif。)

[!TIP]
💡 Memory Master (廣東話口訣):
「抽獎搵 Pattern」: Gibbs Sampling 似抽籤，抽吓抽吓就搵到隱藏嘅規律。

🧬 7. RNA Secondary Structure | RNA 次級結構

Nussinov Algorithm

Maximize base pairs (A-U, G-C). (目標：令到最多鹼基可以成功配對)

[!TIP]
💡 Memory Master (廣東話口訣):
「連連看」: RNA 係一條繩，Nussinov 就係幫佢搵最多「另一半」嚟配對，記住要用 Stack (疊羅漢) 嚟做 Traceback。

🧩 8. Protein Structure & Prediction | 蛋白質結構與預測

Classification | 分類

  • Pfam: Families.
  • SCOP/CATH: Hierarchical (Class, Architecture, Topology, etc.).

Prediction | 預測

  • Homology Modeling: Needs a template. (有模板跟住做)
  • Ab Initio (Rosetta): From scratch. (無模板，由零開始砌)
  • AlphaFold2: Deep learning (The current King).

[!TIP]
💡 Memory Master (廣東話口訣):
「照抄 vs 盲摸」: Homology 係「照抄」（跟 Template），Ab Initio 係「盲摸摸」（Rosetta 逐塊拼），AlphaFold2 係「人工智能大師」。
