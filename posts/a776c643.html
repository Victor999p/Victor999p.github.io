Background Concepts | 背景概念

Even and Odd Components | 偶數與奇數分量

For a continuous-time signalx(t)x(t)x(t):

  • Even component | 偶數分量:xe(t)=x(t)+x(−t)2x_e(t) = \frac{x(t) + x(-t)}{2}xe​(t)=2x(t)+x(−t)​
  • Odd component | 奇數分量:xo(t)=x(t)−x(−t)2x_o(t) = \frac{x(t) - x(-t)}{2}xo​(t)=2x(t)−x(−t)​

For a discrete-time signalx[n]x[n]x[n]:

  • Even component | 偶數分量:xe[n]=x[n]+x[−n]2x_e[n] = \frac{x[n] + x[-n]}{2}xe​[n]=2x[n]+x[−n]​
  • Odd component | 奇數分量:xo[n]=x[n]−x[−n]2x_o[n] = \frac{x[n] - x[-n]}{2}xo​[n]=2x[n]−x[−n]​

A signal is even | 偶數 ifx(t)=x(−t)x(t) = x(-t)x(t)=x(−t) orx[n]=x[−n]x[n] = x[-n]x[n]=x[−n], and odd | 奇數 ifx(t)=−x(−t)x(t) = -x(-t)x(t)=−x(−t) orx[n]=−x[−n]x[n] = -x[-n]x[n]=−x[−n].

Periodicity | 週期性

  • A continuous-time signal | 連續時間信號x(t)x(t)x(t) is periodic if there exists aT>0T > 0T>0 such thatx(t+T)=x(t)x(t + T) = x(t)x(t+T)=x(t). The smallest suchTTT is the fundamental period | 基本週期.
  • A discrete-time signal | 離散時間信號x[n]x[n]x[n] is periodic if there exists an integerN>0N > 0N>0 such thatx[n+N]=x[n]x[n + N] = x[n]x[n+N]=x[n]. The smallest suchNNN is the fundamental period | 基本週期.
  • For sums of sinusoids, periodicity depends on whether the periods of individual components are commensurate (i.e., their ratio is rational).
  • For continuous-time sinusoidscos⁡(ωt)\cos(\omega t)cos(ωt) orsin⁡(ωt)\sin(\omega t)sin(ωt), the period isT=2πωT = \frac{2\pi}{\omega}T=ω2π​.
  • For discrete-time sinusoidscos⁡(ωn)\cos(\omega n)cos(ωn) orsin⁡(ωn)\sin(\omega n)sin(ωn), the signal is periodic ifω2π=kN\frac{\omega}{2\pi} = \frac{k}{N}2πω​=Nk​ for integerskkk andNNN, with periodNNN.

Trigonometric Properties | 三角函數性質

  • cos⁡(−t)=cos⁡(t)\cos(-t) = \cos(t)cos(−t)=cos(t) (even | 偶數)
  • sin⁡(−t)=−sin⁡(t)\sin(-t) = -\sin(t)sin(−t)=−sin(t) (odd | 奇數)
  • These apply similarly for discrete-time signals withnnn.

Signal 1:x(t)=2cos⁡(5t)+3sin⁡(3t)x(t) = 2\cos(5t) + 3\sin(3t)x(t)=2cos(5t)+3sin(3t)

Even and Odd Components | 偶數與奇數分量

Computex(−t)x(-t)x(−t):
x(−t)=2cos⁡(5(−t))+3sin⁡(3(−t))=2cos⁡(−5t)+3sin⁡(−3t)=2cos⁡(5t)−3sin⁡(3t)x(-t) = 2\cos(5(-t)) + 3\sin(3(-t)) = 2\cos(-5t) + 3\sin(-3t) = 2\cos(5t) - 3\sin(3t)x(−t)=2cos(5(−t))+3sin(3(−t))=2cos(−5t)+3sin(−3t)=2cos(5t)−3sin(3t)

xe(t)=x(t)+x(−t)2=[2cos⁡(5t)+3sin⁡(3t)]+[2cos⁡(5t)−3sin⁡(3t)]2=4cos⁡(5t)2=2cos⁡(5t)x_e(t) = \frac{x(t) + x(-t)}{2} = \frac{[2\cos(5t) + 3\sin(3t)] + [2\cos(5t) - 3\sin(3t)]}{2} = \frac{4\cos(5t)}{2} = 2\cos(5t)xe​(t)=2x(t)+x(−t)​=2[2cos(5t)+3sin(3t)]+[2cos(5t)−3sin(3t)]​=24cos(5t)​=2cos(5t)

Odd component | 奇數分量:
xo(t)=x(t)−x(−t)2=[2cos⁡(5t)+3sin⁡(3t)]−[2cos⁡(5t)−3sin⁡(3t)]2=6sin⁡(3t)2=3sin⁡(3t)x_o(t) = \frac{x(t) - x(-t)}{2} = \frac{[2\cos(5t) + 3\sin(3t)] - [2\cos(5t) - 3\sin(3t)]}{2} = \frac{6\sin(3t)}{2} = 3\sin(3t)xo​(t)=2x(t)−x(−t)​=2[2cos(5t)+3sin(3t)]−[2cos(5t)−3sin(3t)]​=26sin(3t)​=3sin(3t)

Verification | 驗證:
xe(t)+xo(t)=2cos⁡(5t)+3sin⁡(3t)=x(t)x_e(t) + x_o(t) = 2\cos(5t) + 3\sin(3t) = x(t)xe​(t)+xo​(t)=2cos(5t)+3sin(3t)=x(t)

Periodicity | 週期性

The signal is a sum of two sinusoids:

  • First term:2cos⁡(5t)2\cos(5t)2cos(5t), angular frequencyω1=5\omega_1 = 5ω1​=5, periodT1=2π5T_1 = \frac{2\pi}{5}T1​=52π​.
  • Second term:3sin⁡(3t)3\sin(3t)3sin(3t), angular frequencyω2=3\omega_2 = 3ω2​=3, periodT2=2π3T_2 = \frac{2\pi}{3}T2​=32π​.

For the sum to be periodic, the periods must be commensurate:
mT1=nT2  ⟹  m⋅2π5=n⋅2π3  ⟹  m5=n3  ⟹  3m=5nm T_1 = n T_2 \implies m \cdot \frac{2\pi}{5} = n \cdot \frac{2\pi}{3} \implies \frac{m}{5} = \frac{n}{3} \implies 3m = 5nmT1​=nT2​⟹m⋅52π​=n⋅32π​⟹5m​=3n​⟹3m=5n
Since 3 and 5 are coprime, the smallest integers arem=5m = 5m=5,n=3n = 3n=3. The fundamental period is:
T=mT1=5⋅2π5=2πorT=nT2=3⋅2π3=2πT = m T_1 = 5 \cdot \frac{2\pi}{5} = 2\pi \quad \text{or} \quad T = n T_2 = 3 \cdot \frac{2\pi}{3} = 2\piT=mT1​=5⋅52π​=2πorT=nT2​=3⋅32π​=2π

Rewrite using the product-to-sum identity:
cos⁡(a)sin⁡(b)=12[sin⁡(a+b)−sin⁡(a−b)]\cos(a) \sin(b) = \frac{1}{2} [\sin(a + b) - \sin(a - b)]cos(a)sin(b)=21​[sin(a+b)−sin(a−b)]

Periodicity | 週期性

Using the product-to-sum form:
x(t)=12sin⁡((5π+3)t)−12sin⁡((5π−3)t)x(t) = \frac{1}{2} \sin((5\pi + 3)t) - \frac{1}{2} \sin((5\pi - 3)t)x(t)=21​sin((5π+3)t)−21​sin((5π−3)t)

  • First term:sin⁡((5π+3)t)\sin((5\pi + 3)t)sin((5π+3)t), angular frequencyω1=5π+3\omega_1 = 5\pi + 3ω1​=5π+3, periodT1=2π5π+3T_1 = \frac{2\pi}{5\pi + 3}T1​=5π+32π​.
  • Second term:sin⁡((5π−3)t)\sin((5\pi - 3)t)sin((5π−3)t), angular frequencyω2=5π−3\omega_2 = 5\pi - 3ω2​=5π−3, periodT2=2π5π−3T_2 = \frac{2\pi}{5\pi - 3}T2​=5π−32π​.

Check if periods are commensurate:
T1T2=2π5π+32π5π−3=5π−35π+3\frac{T_1}{T_2} = \frac{\frac{2\pi}{5\pi + 3}}{\frac{2\pi}{5\pi - 3}} = \frac{5\pi - 3}{5\pi + 3}T2​T1​​=5π−32π​5π+32π​​=5π+35π−3​
Approximate:
5π≈15.70796,5π−3≈12.70796,5π+3≈18.70796,12.7079618.70796≈0.6792455\pi \approx 15.70796, \quad 5\pi - 3 \approx 12.70796, \quad 5\pi + 3 \approx 18.70796, \quad \frac{12.70796}{18.70796} \approx 0.6792455π≈15.70796,5π−3≈12.70796,5π+3≈18.70796,18.7079612.70796​≈0.679245
This ratio is not rational due to the presence ofπ\piπ. Thus, the signal is non-periodic.
